#SCHOOL MANAGEMENT SYSTEM(OOP)

#Description
A python console application built using Object Oriented Programming(OOP)

#The System allows users to:
1. Add Students
2. Add Teachers
3. View Everyone
4. Count People
5. Search for a Person
6. Exit the Program

#OOP Concepts used
1. Classes and Objects
2. Constructors(init)
3. Inheritance
4. Method Overriding
5. Polymorphism
6. Abstraction

#Technologies
- Python 3
  
#How to Run
1. Open a terminal
2. Navigate to the projector
3. Run
python school_system.py
